Prerequisites
Ensure you have the following installed on your system:
Node.js (8.19.2 or later)
npm (Node package manager, comes with Node.js)



Install Dependencies 
Mediapipe: @mediapipe/drawing_utils, @mediapipe/pose, @mediapipe/tasks-vision, @mediapipe/camera_utils, @mediapipe/control_utils, @mediapipe/control_utils_3d
MUI (Material-UI): @mui/icons-material, @mui/material
React Router DOM: react-router-dom
React WebCam: react-webcam
React DOM:  react-dom
React Scripts: react-scripts
Web Vitals: web-vitals


Running Both Servers Simultaneously
To run both the frontend and backend servers simultaneously:
Open two terminal windows or tabs:
Use one terminal to run the backend (node index.js from the backend folder).
Use the other terminal to run the frontend (npm start from the dissertation folder).

Running the Backend Server

Navigate to the Backend Folder 	command -      cd backend
2.    Install Backend Dependencies  	command  -     npm install

3.   Run the Backend Server 	command  -    node index.js



Running the Frontend Server

Navigate to the Front-end Folder 	command -      cd dissertation
2.    Install Dependencies  	command  -     npm install

3.   Run the Backend Server 	command  -    npm start
